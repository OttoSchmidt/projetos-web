acos

Descrição

Retorna o arco cosseno de x em radianos.

Declaração

Argumentos

  • x - Este é o valor do ponto flutuante no intervalo [-1,+1].

frexp

Descrição

O valor de retorno é a mantissa e o número inteiro apontado pelo exponent é o expoente. o valor resultante é x = mantissa * 2 ^ expoente.

Declaração

double frexp(double x, int *exponent)

Argumentos

  • x - Este é o valor do ponto flutuante a ser calculado.
  • exponent - Este é o ponteiro para um objeto int onde o valor do expoente deve ser armazenado.

Valor de retorno

Esta função retorna a fração normalizada. Se o argumento x não for zero, a fração normalizada será x vezes uma potência de dois e seu valor absoluto estará sempre no intervalo de 1/2 (inclusivo) a 1 (exclusivo). Se x for zero, então a fração normalizada é zero e zero é armazenado em exp.
