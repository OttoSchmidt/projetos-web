MACROS

NULL

stdio, stdlib, string

 Possui um valor de um ponteiro vazio constante

BUFSIZ

stdio

 Representa o tamanho de um buffer utilizado na função setbuf

EOF

stdio

 Indica que o end-of-file (fim do arquivo) foi atingido e possui um valor negativo inteiro

FOPEN_MAX

stdio

 Representa o máximo de arquivos que o sistema pode garantir que estarão abertos simultaneamente

FILENAME_MAX

stdio

 Representa o tamanho de uma string para guardar o maior tamanho possível para um nome de arquivo. Se a implementação não possui limites, então esse valor deve ser utilizado como o máximo.

l_tmpnam

stdio

 Representa o tamanho da maior string utilizada para guardar o maior nome temporário de um arquivo criado pela função tmpnam

TMP_MAX

stdio

 Representa o número máximo de arquivos únicos que a função tmpnam pode criar

stder, stdin e stdout

stdio

 Representa ponteiros para variáveis FILE que correspondem a ‘standard erro’ (erro padrão), ‘standard input’ (entrada padrão) e ‘standard output’ (saída padrão)

EXIT_FAILURE

stdlib

 Valor de retorno de função em caso de falha

EXIT_SUCCESS

stdlib

 Valor de retorno de função em caso de sucesso

RAND_MAX

stdlib

 Representa o valor máximo retornado pela função rand

MB_CUR_MAX

stdlib

 Representa o número máximo de bytes de um caracter multi-byte. Ele não pode ser maior que MB_LEN_MAX

HUGE_VAL

math

 Utilizada quando o resultado de uma função não possa ser representado por um número de ponto flutuante. Se o valor de retorno for muito grande, a função define errno como ERANGE e retorna um valor gigante nomeado pela macro. Porém, se a magnitude for muito pequena, será retornado zero
